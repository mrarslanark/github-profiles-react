# Search GitHub Profiles

A React project which lets user search for GitHub Profiles and details regarding the user. This is a practice project.

## Technologies

- React

## Development

- Clone repository
- Install dependencies
  - `npm install`
- Run project locally
  - `npm start`
- Build project for production
  - `npm run build`
